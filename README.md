# beauty_website_creator

## section

构建美女图片站，包含以下几个部分：

- 爬虫：从美女图片站点爬取图片
- 色情检测：通过色情监测去掉色情图片
- 水印：去掉原始水印，增加自己的水印
- wordpress批量上传：将最终图片批量上传到wordpress中

具体操作见代码：只是简单写一下脚本，了解其中过程，没有形成系统。
